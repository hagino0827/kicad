Official Parts (承認済みパーツ)

このフォルダはチームで承認されたフットプリントを置く場所です。

運用ルール:

- 新規フットプリントはブランチ→PR→レビュー→マージのワークフローで追加すること。
- PR には「データシートの参照」「ドリル径/ランド径の確認」「3Dモデル割当の有無」を必須チェック項目とすること。
- フットプリントは1部品1ファイル（`.kicad_mod`）で管理すること。

ファイル配置例:

- `part_name.kicad_mod` - フットプリント本体
- 3Dモデルは `../../3dmodels/` 以下に配置し、フットプリントでパス変数を使って参照すること。

管理者はこのフォルダ内のファイルのみを公式パーツとして設計で使用する。
